# PALM-mGWAS

`PALM-mGWAS` is an R package for microbiome genome-wide association studies based on the `PALM` framework. It supports the main steps of an mGWAS analysis workflow, including input checking, null model fitting, per-study summary statistic generation, meta-analysis across studies, and result reporting. Current genotype input support is limited to PLINK and VCF.

This repository contains the package source code, command-line scripts, example input/output files, and supporting workflow files for running PALM-mGWAS analyses.

Basic package information:

- Package name: `PALMmGWAS`
- Version: `1.0`
- Language: `R`
- License: `GPL (>= 2)`
